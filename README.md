# Agri_Contaminants
All raw data and code for analyses associated with manuscript "Crop-specific and microbially-mediated impacts of alternative agricultural amendments on plant performance"
Ippolito et al. (2026)

## 📁 Project Structure

- **data/**
  Raw and processed datasets used in the analysis.

- **scripts/**  
  R scripts for cleaning, analysis, and visualization.

- **figures/**  
  Output plots and graphics generated by the scripts.

- **docs/**  
  Project documentation, reports, and notes.

- **results/**  
  Summary tables, statistical outputs, and final results.

  - **data/**
  - **raw/** — unmodified input files  
  - **clean/** — cleaned data used in analysis
